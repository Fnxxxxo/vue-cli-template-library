# vue-cli-template-library

> Template for developing open-source vue.js libraries

> Rollup + Jest + Babel + Storybook + TravisCI + SemanticRelease

## Usage

```bash
$ npm install -g vue-cli
$ vue init julon/vue-lib-template my-project
$ cd my-project
$ npm install
$ npm run storybook
```
